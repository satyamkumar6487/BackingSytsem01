# Banking System (Core Java)

This is a console-based Banking System project developed using Core Java.
The project demonstrates Object-Oriented Programming concepts such as
Encapsulation, Abstraction, Exception Handling, and Service-based design.

## Features
- Open New Account
- Deposit Money
- Withdraw Money
- Transfer Money
- View Account Statement
- Input Validation
- Custom Exception Handling

## Technologies Used
- Java
- OOP Concepts
- Collections
- IntelliJ IDEA

## Project Structure
- service → Business logic
- repository → Data storage logic
- model → Account related classes
- validation → Input validation
- exception → Custom exceptions

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run the `Main` class
4. Use console menu to perform operations

## Learning Outcome
- Strong understanding of Java OOP
- Service & Repository pattern
- Clean code structure
- Real-world banking logic implementation
